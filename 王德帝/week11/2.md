| 对比项       | 纯文本提问                                    | 文本 + 图像提问                            |
| --------- | ---------------------------------------- | ------------------------------------ |
| **输入模态**  | 仅文本                                      | 文本 + 图像                              |
| **编码方式**  | Text Embedding (如 bge-large-zh、e5-large) | 双模态编码（CLIP、BLIP-2、MiniCPM-V 等）       |
| **检索空间**  | 文本向量空间                                   | 多模态联合向量空间                            |
| **检索策略**  | 仅语义检索文本内容                                | 同时计算文本相似度与视觉相似度，或进行跨模态检索             |
| **上下文构建** | 拼接相似文本片段                                 | 拼接文本 + 图像描述 + 表格摘要                   |
| **生成模型**  | LLM（纯语言模型）                               | MLLM（多模态大模型，如 GPT-4V、Gemini、Qwen-VL） |
| **应用场景**  | 一般问答、知识问答                                | 视觉问答、图表分析、图片理解                       |
