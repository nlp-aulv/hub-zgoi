# 统一存储方案

结构：

```python
# 主键
id 
# 多模态向量
text_vector # 文件向量
image_vector # 图片向量
table_vector # 表格向量
# 元数据
modality # 模态：文本，图片，表格,...
doc_id # 文件id
chunk_id # 文本的chunk id
file_type # 文本，表格，图片
content # 文本或者表格的具体内容
# 时间
create_time # 创建时间
update_time # 更新时间
```

设计优点：

1. 易于添加新模态

2. 天然有序，无需后续多模态重排序

3. 一次向量相似度计算
